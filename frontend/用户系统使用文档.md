# 用户系统使用文档

## 🎉 功能概述

已成功实现完整的用户认证系统，包括：
- ✅ 学号+密码登录
- ✅ 用户注册
- ✅ JWT令牌认证
- ✅ 记住登录状态

- ✅ 退出登录
- ✅ 密码强度验证

## 📝 API接口文档

### 1. 用户注册
**POST** `/api/auth/register`

**请求体**:
```json
{
  "studentId": "2021002",
  "username": "李四",
  "password": "123456",
  "confirmPassword": "123456",
  "email": "lisi@example.com",
  "phone": "13800138000"
}
```

**响应**:
```json
{
  "success": true,
  "message": "注册成功",
  "user": {
    "id": 2,
    "studentId": "2021002",
    "username": "李四",
    "email": "lisi@example.com",
    "role": "STUDENT"
  }
}
```

### 2. 用户登录
**POST** `/api/auth/login`

**请求体**:
```json
{
  "studentId": "2021001",
  "password": "123456",
  "rememberMe": false
}
```

**响应**:
```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "tokenType": "Bearer",
  "expiresIn": 86400000,
  "user": {
    "id": 1,
    "studentId": "2021001",
    "username": "张三",
    "email": "zhangsan@example.com",
    "role": "STUDENT"
  }
}
```

### 3. 获取当前用户信息
**GET** `/api/auth/me`

**请求头**:
```
Authorization: Bearer {token}
```

**响应**:
```json
{
  "success": true,
  "user": {
    "id": 1,
    "studentId": "2021001",
    "username": "张三",
    "email": "zhangsan@example.com",
    "role": "STUDENT"
  }
}
```

### 4. 刷新令牌
**POST** `/api/auth/refresh`

**请求体**:
```json
{
  "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```

### 5. 修改密码
**POST** `/api/auth/change-password`

**请求头**:
```
Authorization: Bearer {token}
```

**请求体**:
```json
{
  "oldPassword": "123456",
  "newPassword": "654321"
}
```

### 6. 退出登录
**POST** `/api/auth/logout`

## 🎨 前端使用示例

### 在组件中使用用户状态

```vue
<script setup lang="ts">
import { useUserStore } from '@/stores/user'

const userStore = useUserStore()

// 检查登录状态
if (userStore.isLoggedIn) {
  console.log('用户已登录:', userStore.user?.username)
}

// 登录
async function login() {
  const result = await userStore.login('2021001', '123456', false)
  if (result.success) {
    console.log('登录成功')
  }
}

// 退出
async function logout() {
  await userStore.logout()
}
</script>

<template>
  <div>
    <div v-if="userStore.isLoggedIn">
      <p>欢迎, {{ userStore.user?.username }}</p>
      <button @click="logout">退出</button>
    </div>
    <div v-else>
      <button @click="login">登录</button>
    </div>
  </div>
</template>
```

## 🔒 安全特性

### 1. 密码加密
- 使用 BCrypt 算法加密存储
- 密码强度验证（6-20位）

### 2. JWT令牌
- 访问令牌有效期：24小时
- 刷新令牌有效期：7天
- 令牌包含用户学号和角色信息

### 3. 登录保护
- 5次登录失败后锁定账号30分钟
- 记录所有登录尝试（成功/失败）
- 记录IP地址和User-Agent

### 4. CORS配置
- 允许前端跨域访问
- 配置在 `SecurityConfig.java`

## 🛠️ 配置说明

### JWT配置

> **安全改进**: JWT 密钥已迁移至环境变量，不再硬编码在配置文件中。

**配置方式**: 编辑 `backend/.env` 文件：
```bash
JWT_SECRET=你的64位以上强随机字符串
```

生成安全密钥：
```bash
openssl rand -base64 64
```

**application.yml 引用**:
```yaml
jwt:
  secret: ${JWT_SECRET}
  expiration: 86400000     # 访问令牌 24小时
  refresh-expiration: 604800000  # 刷新令牌 7天
```

### 前端API配置 (frontend/src/config.ts)
```typescript
export const API_CONFIG = {
  baseURL: 'http://localhost:8080',
  timeout: 10000
}
```

## 📊 数据库

系统使用 **Spring Data JPA + MySQL 8.0**，表结构由 JPA 自动创建/更新（`ddl-auto: update`）。

### 核心表结构

| 表名 | 说明 |
|------|------|
| `users` | 用户表（学号、密码BCrypt哈希、角色、登录统计） |
| `login_logs` | 登录日志（IP、User-Agent、成功/失败/锁定状态） |
| `books` | 书籍表（含库存字段 totalCopies / availableCopies） |
| `borrow_records` | 借阅记录（6状态枚举、续借、逾期罚款） |
| `reservation_records` | 预约记录（排队位置、过期管理） |
| `notification_records` | 通知记录（类型、已读标记） |

## 🐛 常见问题

### 1. 登录后刷新页面丢失状态
**解决**: 已实现自动恢复，token存储在localStorage中

### 2. CORS错误
**解决**: 检查 `SecurityConfig.java` 中的CORS配置，确保包含前端地址

### 3. Token过期
**解决**: 使用refreshToken自动刷新，或重新登录

### 4. 密码验证失败
**解决**: 确保密码长度6-20位
