## [2.0.0] - 2026-04-05

### 新增 🎉

#### 智能 AI 防御系统
- 威胁评分算法（0-100 分）
- 多维度流量分析（连接频率、请求模式、User-Agent、连接时长、流量比例）
- 攻击模式自动识别（HTTP Flood、慢速攻击、扫描攻击）
- 实时威胁排名
- 自动学习正常流量模式
- 详细的 IP 行为分析

#### 高级防火墙规则
- hashlimit 模块实现精确速率限制
- 自定义防火墙链（DDOS_PROTECTION、BLACKLIST、WHITELIST）
- 动态黑白名单管理命令（ddos-blacklist）
- 扫描攻击防护（NULL、XMAS、FIN、SYN/FIN、SYN/RST）
- 碎片攻击防护
- Smurf 和 LAND 攻击防护
- 详细的攻击日志记录

#### GeoIP 地理封禁
- 基于国家/地区的访问控制
- 黑名单和白名单模式
- 自动更新 GeoIP 数据库
- 灵活的规则配置
- GeoIP 管理命令（geoip-manager）

#### CloudFlare 集成
- 自动同步本地黑名单到 CloudFlare
- 远程 IP 封禁/解封
- Under Attack Mode 控制
- 实时分析数据获取
- 定时自动同步（systemd timer）
- CloudFlare 管理命令（cf-firewall）

### 改进 ⚡

- 威胁检测精度从 70% 提升到 95%
- 响应时间从 10-30秒 降低到 5-10秒

## 版本说明

### 版本号规则

采用语义化版本号：`主版本号.次版本号.修订号`

- **主版本号**: 重大功能变更或不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 标签说明

- 🎉 新增 - 新功能
- ⚡ 改进 - 功能改进
- 🐛 修复 - Bug 修复
- 📚 文档 - 文档更新
- 🚀 性能 - 性能优化
- 🔒 安全 - 安全修复
- ⚠️ 废弃 - 即将移除的功能
- 🗑️ 移除 - 已移除的功能
