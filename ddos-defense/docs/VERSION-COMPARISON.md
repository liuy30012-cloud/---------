## 💡 升级建议

### 从基础版升级到增强版

**步骤 1: 备份现有配置**
```bash
sudo iptables-save > /backup/iptables-basic.rules
sudo cp -r /etc/nginx /backup/nginx-basic
```

**步骤 2: 部署增强功能**
```bash
# 高级防火墙
sudo ./scripts/advanced-firewall.sh

# 智能防御
sudo python3 monitoring/intelligent-defense.py &

# GeoIP（可选）
sudo ./scripts/geoip-setup.sh

# CloudFlare（可选）
sudo ./scripts/cloudflare-integration.sh
```

**步骤 3: 测试验证**
```bash
sudo ./scripts/status-check.sh
tail -f /var/log/intelligent-ddos.log
```

**步骤 4: 监控观察**
- 观察 1-2 天
- 检查误封情况
- 调整参数
