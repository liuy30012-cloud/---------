## 🎯 统计算法

### 热门书籍算法
```java
// 统计每本书的借阅次数
Map<Long, Long> bookBorrowCount = allRecords.stream()
    .collect(Collectors.groupingBy(
        BorrowRecord::getBookId,
        Collectors.counting()
    ));

// 按借阅次数降序排序，取Top N
return bookBorrowCount.entrySet().stream()
    .sorted(Map.Entry.<Long, Long>comparingByValue().reversed())
    .limit(limit)
    .map(entry -> convertToDTO(entry))
    .collect(Collectors.toList());
```

### 借阅趋势算法
```java
// 按日期分组统计借阅和归还
Map<LocalDate, Long> borrowByDate = records.stream()
    .collect(Collectors.groupingBy(
        r -> r.getBorrowDate().toLocalDate(),
        Collectors.counting()
    ));

// 生成最近N天的完整趋势数据
for (int i = days - 1; i >= 0; i--) {
    LocalDate date = LocalDate.now().minusDays(i);
    Long borrowCount = borrowByDate.getOrDefault(date, 0L);
    trends.add(new BorrowTrendDTO(date, borrowCount));
}
```

### 分类统计算法
```java
// 按分类分组
Map<String, List<Book>> booksByCategory = allBooks.stream()
    .filter(book -> book.getCategory() != null)
    .collect(Collectors.groupingBy(Book::getCategory));

// 计算每个分类的统计数据
return booksByCategory.entrySet().stream()
    .map(entry -> {
        long totalBooks = entry.getValue().size();
        long borrowedBooks = calculateBorrowedBooks(entry.getValue());
        double borrowRate = (borrowedBooks * 100.0 / totalBooks);

    })
    .collect(Collectors.toList());
```

### 用户画像算法
```java
// 统计用户最喜欢的分类
Map<String, Long> categoryCount = userRecords.stream()
    .map(BorrowRecord::getBookId)
    .map(bookId -> getBook(bookId))
    .filter(Objects::nonNull)
    .collect(Collectors.groupingBy(Book::getCategory, Collectors.counting()));

String favoriteCategory = categoryCount.entrySet().stream()
    .max(Map.Entry.comparingByValue())
    .map(Map.Entry::getKey)
    .orElse("未知");

// 计算平均借阅天数
double averageBorrowDays = userRecords.stream()
    .filter(r -> r.getReturnDate() != null)
    .mapToLong(r -> ChronoUnit.DAYS.between(r.getBorrowDate(), r.getReturnDate()))
    .average()
    .orElse(0.0);
```
