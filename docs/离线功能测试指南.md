# 离线功能测试指南

## 测试环境
- 前端服务器: http://localhost:5175
- 后端服务器: http://localhost:8080 (需要单独启动)

## 测试前准备

### 1. 启动后端服务器
```bash
cd backend
mvn spring-boot:run
```

### 2. 启用 Service Worker (开发环境)
创建 `frontend/.env.local` 文件:
```
VITE_ENABLE_SW=true
```

然后重启前端服务器。

## 测试步骤

### 测试 1: 验证 Service Worker 注册

1. 打开浏览器访问 http://localhost:5175
2. 打开 Chrome DevTools (F12)
3. 切换到 **Application** 标签
4. 左侧菜单选择 **Service Workers**
5. 确认看到 Service Worker 已注册并激活

**预期结果**: 看到 `sw.js` 状态为 "activated and is running"

### 测试 2: 验证 IndexedDB 初始化

1. 在 DevTools 的 **Application** 标签
2. 左侧菜单选择 **Storage > IndexedDB**
3. 展开 `LibraryOfflineDB`

**预期结果**: 看到三个对象存储:
- books
- cache_metadata
- search_cache

### 测试 3: 浏览书籍并缓存

1. 登录系统 (账号: 2021001, 密码: Test123456)
2. 搜索一些书籍
3. 点击查看几本书的详情
4. 在 DevTools 中检查 IndexedDB > LibraryOfflineDB > books

**预期结果**: 看到浏览过的书籍已保存到 IndexedDB

### 测试 4: 离线模式测试

1. 在 DevTools 的 **Network** 标签
2. 勾选 **Offline** 复选框 (模拟离线)
3. 刷新页面

**预期结果**: 
- 页面顶部显示 "离线模式" 提示条
- 可以正常访问首页
- 静态资源从缓存加载

### 测试 5: 离线搜索书籍

1. 保持离线模式
2. 在搜索框输入之前浏览过的书籍名称
3. 点击搜索

**预期结果**: 
- 显示已缓存的书籍结果
- 提示 "离线模式:显示已缓存的书籍"

### 测试 6: 离线查看书籍详情

1. 保持离线模式
2. 点击已缓存的书籍卡片

**预期结果**: 
- 可以查看书籍详情
- 显示基本信息(标题、作者、位置等)
- 提示 "离线模式:显示缓存的书籍信息"
- 借阅、预约按钮被隐藏或禁用

### 测试 7: 离线操作限制

1. 保持离线模式
2. 尝试执行需要网络的操作(如借阅、预约)

**预期结果**: 
- 弹出提示 "需要网络连接"
- 操作被阻止

### 测试 8: 网络恢复

1. 在 DevTools 中取消勾选 **Offline**
2. 观察页面变化

**预期结果**: 
- "离线模式" 提示条消失
- 后台自动更新缓存
- 所有功能恢复正常

### 测试 9: 缓存管理界面

1. 访问设置页面
2. 找到缓存管理组件 (需要先集成到设置页面)

**预期结果**: 
- 显示缓存统计(已缓存书籍数、最后更新时间)
- "立即更新缓存" 按钮可用
- "清空所有缓存" 按钮可用

### 测试 10: 手动更新缓存

1. 在缓存管理界面
2. 点击 "立即更新缓存"

**预期结果**: 
- 显示 "更新中..." 状态
- 更新完成后显示 "缓存更新成功"
- 统计数据更新

### 测试 11: 清空缓存

1. 在缓存管理界面
2. 点击 "清空所有缓存"
3. 确认操作

**预期结果**: 
- 显示确认对话框
- 清空后显示 "缓存已清空"
- 统计数据归零
- IndexedDB 中的数据被清空

## 常见问题排查

### Service Worker 未注册
- 确认浏览器支持 Service Worker
- 确认是 HTTPS 或 localhost 环境
- 检查控制台错误信息
- 确认 `VITE_ENABLE_SW=true` 已设置

### 离线功能不生效
- 确认 Service Worker 已激活
- 检查 IndexedDB 是否正常初始化
- 查看 Network 标签确认请求被拦截
- 检查控制台是否有错误

### 缓存未更新
- 确认网络连接正常
- 检查后端服务器是否运行
- 查看控制台日志
- 尝试手动更新缓存
