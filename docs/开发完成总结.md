## 🎯 实现的功能

### 核心功能
- ✅ 用户注册（学号+密码）
- ✅ 用户登录（JWT认证）

## 🚀 如何使用

### 1. 启动后端
```bash
cd backend
mvn spring-boot:run
```

### 2. 启动前端
```bash
cd frontend
npm install  # 首次运行
npm run dev
```

### 3. 访问系统
- 前端：http://localhost:5173
- 后端：http://localhost:8080

### 4. 测试账号
- 学号：`2021001`
- 密码：`123456`

## 📝 API接口

| 端点 | 方法 | 说明 |
|------|------|------|
| /api/auth/register | POST | 用户注册 |
| /api/auth/login | POST | 用户登录 |
| /api/auth/refresh | POST | 刷新令牌 |
| /api/auth/me | GET | 获取当前用户 |
| /api/auth/change-password | POST | 修改密码 |
| /api/auth/logout | POST | 退出登录 |

## 🔒 安全措施

1. **密码安全**
   - BCrypt加密存储
   - 不可逆加密
   - 密码强度验证

2. **令牌安全**
   - JWT签名验证
   - 短期访问令牌（24小时）
   - 长期刷新令牌（7天）

3. **登录保护**
   - 5次失败锁定30分钟
   - IP地址记录
   - User-Agent记录

4. **数据验证**
   - 前端表单验证
   - 后端Jakarta Validation
   - 全局异常处理
