## 📊 预警算法

### 预警检查逻辑
```java
private InventoryAlertDTO checkBookAlert(Book book, Long borrowCount) {
    int totalCopies = book.getTotalCopies();
    int availableCopies = book.getAvailableCopies();
    int borrowedCopies = totalCopies - availableCopies;

    // 1. 检查是否缺货（所有副本都被借出）
    if (availableCopies == 0 && borrowCount > 0) {
        alertType = "OUT_OF_STOCK";
        alertLevel = "CRITICAL";
        message = "所有副本已借出，建议增加库存";
    }
    // 2. 检查低库存（可借副本 <= 总副本的20%）
    else if (availableCopies > 0 && availableCopies <= totalCopies * 0.2) {
        alertType = "LOW_STOCK";
        alertLevel = "WARNING";
        message = String.format("库存不足，仅剩%d本可借", availableCopies);
    }
    // 3. 检查高需求（借阅次数 >= 10 且可借副本 <= 总副本的30%）
    else if (borrowCount >= 10 && availableCopies <= totalCopies * 0.3) {
        alertType = "HIGH_DEMAND";
        alertLevel = "WARNING";
        message = String.format("热门书籍，借阅%d次，建议增加库存", borrowCount);
    }
}
```

### 预警排序逻辑
```java
// 按预警级别和借阅次数排序
alerts.sort((a, b) -> {
    // 1. 先按预警级别排序（CRITICAL优先）
    int levelCompare = getAlertLevelPriority(a.getAlertLevel()) 
                     - getAlertLevelPriority(b.getAlertLevel());
    if (levelCompare != 0) return levelCompare;
    
    // 2. 再按借阅次数降序排序
    return Long.compare(b.getBorrowCount(), a.getBorrowCount());
});
```
